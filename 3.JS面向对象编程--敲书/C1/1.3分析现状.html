有意思的是，JavaScript必须运行于某种宿主环境中。Web浏览器仅仅是其中一种，JS也完全可以运行在服务器端、桌面以及移动设备中。如今，我们已经可以用JS来实现以下功能。

  • 创建拥有强大而丰富功能的Web应用程序（这种应用程序往往运行在Web浏览器中）。另外还有基于HTML5的许多特性，例如应用缓存、本地存储、本地数据库。无论是线上应用还是离线应用，web应用都可以做得非常强大。
  • 使用.NET和Node.js编写服务器端脚本，或者使用Rhino（这是一种用Java实现的JS引擎）这样的框架来进行编程。
  • 为移动设备编写各种应用程序。借助于PhoneGap及Titanium这样的工具，我们完全可以使用纯js来编写iphone、android或其他平台上应用程序。另外只得一提的是，移动平台FireFox OS的原生编程语言就是JS，HTML，CSS。
  • 使用ActionScript创建富媒体应用（如Flash，Flex）。ActionScrit也是一种基于ECMAScript标准的脚本语言。
  • 编写各种机遇命令行的、用于桌面自动化管理的脚本服务。其自带的宿主环境如Windows Scripting Host及Mac下的WebKit JavaScript Core。
  • 为一些桌面应用程序编写扩展或插件，例如Dreamweaver、Photoshop及大多数浏览器。
  • 使用Mozilla XUIRunner及Adobe Air创建跨操作系统的桌面应用程序。
  • 使用Yahoo！Widgets及Mac Dashboard Widgets等工具包来创建桌面小工具。其中，Yahoo！Widgets还可以在智能电视上运行。

当然，这里列出的远远不是该语言的全部应用。JS应用的确发端于Web页面，但如今几乎可以说是无所不在了。另外，浏览器厂商如今都将运行速度视为产品的竞争优势之一，因此都致力于创建更快地JS引擎。这对于用户与开发者来说无疑是个好消息，并且浙江打开一扇大门——在新的领域，例如在图像、音频及视频处理、游戏开发等方面，JS也必将一展拳脚。
