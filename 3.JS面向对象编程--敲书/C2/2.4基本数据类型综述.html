现在，让我们来快速汇总一下目前所讨论的内容。

    JS语言中有五大基本数据类型：
    • 数字
    • 字符串
    • 布尔值
    • undefined
    • null
  • 任何不属于基本类型的东西都属于对象。
  • 数字类型可以存储的数据包括：正负整数、浮点数、十六进制数与八进制数、指数以及特殊数值NaN、Infinity、-Infinity。
  • 字符串类型存储的是一对引号之间的所有字符。
  • 布尔类型的值只有两个：true、false
  • null类型的值只有一个：null
  • undefined类型的值只有一个：undefined
  • 绝大部分值在转换为布尔类型时都为true，但以下6中falsy值除外：
    • ""
    • null
    • undefined
    • 0
    • NaN
    • false
